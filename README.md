# Keep Kids Private

A campaign website advocating for children's privacy online — protecting kids' data, delaying social media access, and giving families practical tools for the digital age.

## Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS 4
- Mantine (drawer / mobile nav)

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Deploy

Configured for [Vercel](https://vercel.com) via `vercel.json`. Connect the repo and deploy, or:

```bash
npx vercel
```
